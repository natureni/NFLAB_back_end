## 🔧 部署配置

### 环境要求
```bash
Python 3.8+
FastAPI 0.104.1+
SQLAlchemy 2.0+
PostgreSQL/SQLite
Redis (可选)
```

### 启动命令
```bash
# 安装依赖
pip install -r requirements.txt

# 启动服务
python app/main.py
# 或
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

### 访问地址
- **API文档**: http://localhost:8000/docs
- **ReDoc**: http://localhost:8000/redoc
- **健康检查**: http://localhost:8000/health
